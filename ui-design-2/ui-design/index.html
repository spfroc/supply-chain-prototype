🔑 核心差异：普通会员 vs 协议企业会员
对比维度 普通会员 协议企业会员
商品价格 显示通用会员价 显示协议折扣价（红色大字）
"专属"Logo 无 商品卡片右上角，红色文字，左倾 45°
下单权限 按钮置灰，提示"暂无有效协议，请联系客服" 正常下单
专区比价 "我的拿货价"显示"登录后查看" 三列完整比价：原价 / 平台售价 / 拿货价
方案价格 显示方案折扣价（不可下单） min(方案折扣价, 协议折扣价)，标注"协议价更优"
购物车 结算按钮置灰 正常结算
个人中心 "普通会员" + 红色"暂无有效协议"提示 企业名称 + 协议有效期 + "已生效"标签
